# 🌐 Página Web - Sistemas Informáticos

## 📖 Descripción del Proyecto

Este proyecto consiste en el desarrollo de una página web informativa sobre los sistemas informáticos.  
La página fue diseñada utilizando HTML5 y CSS3, incorporando un diseño moderno, animaciones visuales y una navegación dinámica entre secciones.

El objetivo principal es brindar información clara y organizada acerca de los sistemas informáticos, sus componentes, tipos e importancia en la actualidad.

---

# 🎯 Objetivos

- Crear una página web profesional e interactiva.
- Aplicar conocimientos de HTML y CSS.
- Implementar menús de navegación funcionales.
- Diseñar una interfaz moderna y atractiva.
- Utilizar comentarios dentro del código para documentar cada sección.

---

# 🛠️ Tecnologías Utilizadas
